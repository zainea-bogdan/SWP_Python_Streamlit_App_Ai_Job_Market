# Commits Code Rules:

- DONE = if a feature is completely functional or not.
- FIX = a feature is partially done - at the end of the day commit it with this code.
- REPAIRED = after a feature is repaired imediately push the code
- HELP = a feature is broken due to an unknown and complicated error.

ex:

> DONE: completed the functionality of writing text or whatever

- git pull ALWAYS at the start of working session :)
